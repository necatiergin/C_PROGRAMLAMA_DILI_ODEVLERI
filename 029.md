#### Aşağıdaki kod için şu sorulara yanıt vermeye çalışın:

* sentaks hatası var mı?
* tanımsız davranış *(undefined behavior)* var mı?
* ekran çıktısı ne olur?

```
#include <stdio.h>

int main(void)
{
	int x = 0;
	int y = 1;

	printf("%d\n", !!x == !!y > x + 1);
	printf("%d\n", !!y > !y == !!x < !x);
	printf("%d\n", !x + !!x + y - !y - !!y);
}
```
