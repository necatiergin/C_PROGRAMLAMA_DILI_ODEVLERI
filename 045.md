#### Aşağıdaki C programında yorum satırı bulunan yere bir kod eklemeniz isteniyor:

```
#include <stdio.h>

int main()
{
	int x, y, z;

	printf("uc tamsayi giriniz: ");
	scanf("%d%d%d", &x, &y, &z);
	/* code  */

}
```

#### Yazdığınız kod çalıştırıldığında girilen sayıları ekrana şu formatta yazdırmalı:

```
girdi           çıktı
=====        ============
30 10 20     10 < 20 < 30
7 5 7        5 < 7 = 7
2 2 2        2 = 2 = 2
9 6 6        6 = 6 < 9
```
